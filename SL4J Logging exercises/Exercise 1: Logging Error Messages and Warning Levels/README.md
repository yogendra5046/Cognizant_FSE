# Exercise 1 - Logging Error Messages and Warning Levels

## Objective
Implement logging in a Java application using the SLF4J framework.

## Technologies Used
- Java
- Maven
- SLF4J

## Concepts Covered
- Logger
- LoggerFactory
- INFO
- WARN
- ERROR

## Files
- App.java

## Expected Output
The application logs INFO, WARN and ERROR messages successfully.

## Learning Outcome
Learned how to implement structured logging using SLF4J instead of System.out.println().
